## ğŸš¨ Troubleshooting

### "Invalid API Key" Error

**Problem**: The API key doesn't work

**Solutions**:
1. Make sure you copied the entire key (starts with `AIzaSy`)
2. Check for extra spaces at the beginning or end
3. Try generating a new API key
4. Make sure the API key is enabled in Google AI Studio

### "Quota Exceeded" Error

**Problem**: You've hit the free tier limit

**Solutions**:
1. Wait until tomorrow (quota resets daily)
2. Upgrade to paid tier (optional, very cheap)
3. Create a new Google account for a new API key

### "API Key Not Working"

**Problem**: Key was working, now it's not

**Solutions**:
1. Check if the key was deleted in Google AI Studio
2. Verify the key is still enabled
3. Check Google AI Studio for any service outages
4. Try regenerating the key
